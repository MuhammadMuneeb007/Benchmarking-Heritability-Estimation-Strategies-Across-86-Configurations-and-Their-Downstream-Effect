# Column Descriptions

| Column Name         | Description                                                                                       |
|---------------------|---------------------------------------------------------------------------------------------------|
| **Unnamed: 0**      | Index or row number.                                                                              |
| **clump_p1**        | P-value threshold for index SNPs in clumping.                                                     |
| **clump_r2**        | LD r-squared threshold for clumping.                                                              |
| **clump_kb**        | Physical distance threshold (in kilobases) for clumping.                                          |
| **p_window_size**   | Pruning                                  |
| **p_slide_size**    | Pruning                                             |
| **p_LD_threshold**  | Pruning                               |
| **h2**              | Estimated heritability of the trait.                                                              |
| **numberofvariants**| Number of genetic variants (SNPs) used in the analysis.                                           |
| **relatedmatrix**   | Ignore it.                        |
| **data**            | Ignore it.                             |
| **numberofpca**     | Number of principal components used in the analysis.                                              |
| **h2model**         | Model used for heritability estimation.                                                           |
| **clumpprune**      | Indicator of whether clumping/pruning was applied (yes/no).                                       |
