


# Column Explanations for BMI Phenotype Analysis

| Column Name       | Description                                       | Details                                                                 |
|-------------------|---------------------------------------------------|-------------------------------------------------------------------------|
| clump_p1          | Primary significance threshold for clumping       | Ignore it |
| clump_r2          | R-squared threshold for clumping                  | Ignore it |
| clump_kb          | Distance threshold for clumping (in kilobases)    | Ignore it |
| p_window_size     | Window size for SNP selection                     | Ignore it |
| p_slide_size      | Slide size for moving the window                  | Ignore it |
| p_LD_threshold    | Linkage disequilibrium (LD) threshold             | Ignore it |
| h2                | Heritability estimate                             | Proportion of variance in the phenotype that is attributable to genetic variance |
| numberofvariants  | Number of variants                                | Total number of genetic variants (SNPs) considered in the analysis |
| h2model           | Heritability model used                           | Provides the specific heritability precomputed file applied in the analysis |

**0 means those columns are not applicable to this Method.**
